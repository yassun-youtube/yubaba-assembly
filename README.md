# 実行方法

## Macにnasmをインストール

```
brew install nasm
```

## 実行

```
sh assemble.sh
```

# ファイル概要

|ファイル/フォルダ|概要|

| test.s | 実際の湯婆婆コード |

| 02. STDIN | 標準入力をやってみたフォルダ |
